## 再此特别感谢KMS服务器程序的开发者Wind4

## 在你的服务器上，执行如下命令即可：

```shell
wget https://raw.githubusercontent.com/Mr-xn/kms-server-deploy/master/kms-server-deploy.sh && bash kms-server-deploy.sh
```
